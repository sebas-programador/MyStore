# MyApp

## Descripción

Aplicación Mobile desarrollado para la venta de productos que permite ver el listado y detalle de cada producto.

Desarrollada con Ionic 

## Instalación y ejecución<a name="install"></a>

1. Clona el proyecto desde github utilizando el comando
```bash
git clone git@github.com:sebas-programador/MyStore.git
```

1. Luego de clonar el repositorio ingrese al proyecto e instale los paquetes.

```bash
npm install
```

3. Ejecute el proyecto en local

```bash
ionic serve
```

## Manejo de commits en local<a name="commits"></a>

El proyecto utiliza conventional commits para la estandarización <br>
[Conventional commits](https://www.conventionalcommits.org)

Flujo de trabajo:

1. Agregar cambios al stage area
2. Agregar el commit según la estandarización, para esto puede utilizar [conventional commits extension](https://marketplace.visualstudio.com/items?itemName=vivaxy.vscode-conventional-commits)

4. Subir al remoto

```bash
git push
```
